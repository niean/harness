示例维度参考（根据项目需要选择和定制）：

架构边界类：
- UI 层不直接调用底层服务，须通过 Coordinator 或 Manager

- 统一日志框架
- 禁止输出敏感信息
- 错误信息分层（用户提示 vs 开发日志）

资源处理类：
- 资源降采样/压缩规范
- 按需加载 + 缓存策略
- 列表页延迟加载

废弃代码类：
- 未使用的类型/函数/变量/文件
- 无效导入
- 过期注释
