# Skill: 任务总结

触发：AI 自动触发，每次任务（功能迭代、Skill 执行等）完成后自动执行。

## 步骤

### Step 1 -- 收集信息

回顾本次任务的完整过程，收集以下信息。

### Step 2 -- 输出报告

按下方"报告模板"输出任务总结报告，事项可根据实际情况增减。

### Step 3 -- 交付方式

将总结报告以文本形式输出到消息框，供用户查阅；不落盘文件。

### Step 4 -- 输出约束

总结报告必须作为独立的消息输出，不得与 attempt_completion 等工具调用合并在同一个响应中；先输出报告，等用户确认收到后，再调用 attempt_completion 结束任务。

## 报告模板

```
## 任务总结

### 意图识别
本次任务的背景和目标，多个目标时分行列出
1.
2.
...

### 关键步骤
实际执行的关键步骤列表，按时间顺序，多个步骤时分行列出
1.
2.
...

### 结果验收
- 最终交付物：是否达成目标
- 构建/测试：是否通过
- ...

### 合规审计
- 是否遵守 AGENTS.md 约束（代码规范、安全规范、架构边界等）
- 有无违规或豁免
- ...

### 资源消耗
- 耗时（任务开始到结束的时间跨度）
- Token 消耗（如可获取）
- 工具调用次数
- ...

```
