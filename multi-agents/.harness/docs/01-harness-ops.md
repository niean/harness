# Harness 项目维护

## 约束
本文仅供自然人使用，未经人工确认、禁止AI修改。

---

## 原则
- AI原生：以AI为中心、以AI全自动化为目标，人类只是临时的协调器。例如，
    - 规则：人类输入原始规则描述，经AI思考加工后再回填 -- AI回填的规则才是有效规则 
    - 需求：人类输入原始需求描述，经AI理解后生成正式输入 -- AI理解的需求才是有效需求(PRD)
    - 治理：治理是一种非功能性需求
- 大任务分解：分步、分段、分批，子任务
- 上下文管理：在大任务分解的基础上，自动压缩，检查点、文件最小化
